1) What does the man command do? Type in man rm. How do you scroll and get out?
The user manual of any command that we can run on the terminal.
Type D or Control ⌃-D to advance half a page - B or Control ⌃-B to go back one page 

2) Look at the man page for ls. What does the -l flag do? What does the -a flag do?
The -l flag lists files in a list with more detail and the -a lists all files and folders plus hidden ones

3) How do you jump between words in the terminal?
You do this with option left/right.

4) How do you get to the end of a line in terminal?
This is done with control + e.

5) How do you move your cursor to the beginning in terminal?
You do this with control + a.

6) How do you delete a word (without pressing backspace multiple times) in terminal?
You do this with option + delete.

7) What is the difference between a terminal and shell?
Terminal is considered an interface to the shell. Otherwise the wrapper program of shell.
Shell handles commands.

8) What is an absolute path?
Absolute path gives the complete location for a file or folder.

9) What is an relative path?
Relative path is the current location.

10) What is a flag? Give three examples of flags you have used.
A flag adds more functionality to an existing command. Examples are -a, -l, and -r. 

11) What do the r and f flags do with the rm command?
r remove folders and recursively go through each folder and file and f will force removal
